# Archived Conda Environment Files

This folder contains conda environment files that are **not supported** in the current release.

## Contents

| File | Platform | Compatible app version |
|---|---|---|
| `environment_MacOS_arm64.yml` | macOS | v1.1.0 |
| `environment_WindowsOS.yml` | Windows | v1.1.0 |

## Why are these files here?

Version **1.2.0rc1** of GW Data Plotter is distributed as source code only, with initial support limited to **Linux**.  

Support for macOS and Windows will be restored in upcoming releases after additional testing on those platforms.

These files are kept for reference and will be moved back to the main `conda_environments/` folder once the respective platforms are officially supported again.
